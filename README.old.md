# KindKitchens Frontend

This is the frontend for KindKitchens, built with React, Vite, and styled-components.

## Features

- User authentication (Donor, NGO, Admin)
- Donation posting and management for donors
- Donation browsing and claiming for NGOs
- Admin dashboard for monitoring the system
- Responsive design

## Technologies Used

- React
- Vite
- React Router
- Axios for API requests
- Styled Components for styling

## Setup Instructions

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm run dev
   ```

3. The app will be available at http://localhost:3000

## Folder Structure

- `src/components` - Reusable components
- `src/pages` - Page components
- `src/assets` - Static assets
- `src/App.jsx` - Main app component with routing
- `src/main.jsx` - Entry point

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
