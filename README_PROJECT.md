# KindKitchens Frontend

Frontend for KindKitchens - A platform to reduce food wastage by connecting food donors with NGOs.

## Tech Stack
- React
- Vite
- Styled Components
- Axios

## Features
- User authentication (Donor/NGO/Admin)
- Donation posting and management
- Food quality assessment
- Location-based NGO matching
- Analytics dashboard

## Deployment
This frontend is configured to work with the deployed backend at https://kindkitchens.onrender.com/

## Development

### Prerequisites
- Node.js
- npm

### Installation
```bash
npm install
```

### Running the Application
```bash
npm run dev
```

### Building for Production
```bash
npm run build
```
